# SpringBoot VS SpringMVC

| SpringBoot | SpringMVC |
| --- | --- |

| 零配置 `yml` | 配置繁琐 `xml`  |
| 集成了很多中间件`*.stater` | |
| 内置 `tomcat` | 外置置 `tomcat` |
